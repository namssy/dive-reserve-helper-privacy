개인정보 처리방침 — Dive Reserve Helper

최종 업데이트: 2026-05-31

Dive Reserve Helper("본 확장 프로그램")는 딥스테이션(deepstation.kr) 프리다이빙 예약을 돕는 비공식 Chrome 확장 프로그램입니다.

수집하는 정보

본 확장 프로그램은 어떠한 개인정보도 수집·저장·외부 전송하지 않습니다.

  • 사용자의 브라우저에 데이터를 저장하지 않습니다(로컬 스토리지/쿠키 생성 없음).
  • 제3자 서버로 어떠한 데이터도 전송하지 않습니다.
  • 분석(애널리틱스), 광고, 추적 도구를 사용하지 않습니다.

데이터 사용 방식

본 확장 프로그램은 예약 현황을 표시하기 위해 사용자가 deepstation.kr에 로그인한 상태에서, 해당 사이트의 예약 조회 API(deepstation.kr/rez/ajax.dayinfo.php)로 인증된 요청(브라우저 쿠키 포함)을 보냅니다.

  • 이 요청은 사용자의 브라우저와 deepstation.kr 사이에서만 이루어집니다.
  • 응답 데이터(부이/인원 잔여 현황)는 화면 표시 용도로만 메모리에서 사용되며, 저장되지 않습니다.
  • 예약 폼 자동 채우기는 deepstation.kr 페이지 내 입력 필드에만 값을 채웁니다.

권한 사용 사유

권한사유
host_permissions: https://deepstation.kr/*예약 페이지에 현황 패널을 표시하고 예약 폼을 자동으로 채우기 위해 필요합니다.

본 확장 프로그램은 deepstation.kr 외 다른 사이트에서는 동작하지 않습니다.

문의

문의 사항은 저장소 이슈를 통해 전달해 주세요.

이 확장 프로그램은 딥스테이션(deepstation.kr)의 공식 도구가 아닌 비공식(Unofficial) 도구입니다.
