개인정보 처리방침 — Dive Reserve Helper

최종 업데이트: 2026-06-26

⚠️ 본 확장 프로그램은 딥스테이션(deepstation.kr)·파라다이브(paradive.co.kr)의 공식 도구가 아닌 비공식(Unofficial) 도구이며, 두 업체와 아무런 관련이 없습니다.

Dive Reserve Helper("본 확장 프로그램")는 딥스테이션(deepstation.kr) 및 파라다이브(paradive.co.kr) 프리다이빙 예약을 돕는 비공식 Chrome 확장 프로그램입니다. 본 방침은 확장 프로그램이 어떤 정보를 어디에 보관하고 어떻게 사용하는지 설명합니다.

한눈에 보기

  • 본 확장 프로그램은 별도의 서버를 운영하지 않으며, 사용자가 입력한 개인정보(이름·생년월일·연락처·병력이력·서명 등)를 외부로 전송하거나 판매·광고에 사용하지 않습니다.
  • 다만 익명 사용량 통계를 위해 어떤 페이지·기능이 쓰이는지에 관한 비식별 정보를 Google Analytics(제3자)로 전송합니다. 개인을 식별할 수 있는 정보는 전송하지 않습니다.
  • 서약서/동의서 자동 채우기 기능을 위해 일부 정보(이름·생년월일·성별·연락처·등급·소속·병력이력·서명·동의 여부 등)를 사용자 기기(브라우저)에만 저장합니다.
  • 구글 캘린더 저장 기능은 사용자가 직접 버튼을 눌렀을 때, 해당 예약 정보를 구글 캘린더(제3자)로 전달합니다.

로컬에 저장되는 정보 — 서약서/동의서 자동 채우기

딥스테이션 강사서약·이용규정서약 페이지, 그리고 파라다이브 동의서(pledge01.php·pledge02.php) 페이지에서 매번 반복 입력하는 항목을 다음 작성 시 자동으로 채워 넣기 위해, 사용자가 입력한 값을 사용자 브라우저 안(chrome.storage.local)에만 저장합니다. (딥스테이션과 파라다이브는 서로 다른 저장 키로 분리 보관합니다.)

  • 저장 대상은 한정된 항목으로 고정되어 있습니다(이 목록 밖의 필드는 저장하지 않습니다).
    • 딥스테이션: 생년월일, 성별, 강사 등급, 라이선스 등급, 병력이력(및 상세), 서명 이미지
    • 파라다이브: 이름, 생년월일, 성별, 휴대폰, 라이선스 등급, 병력이력(및 상세), 서명 이미지, 강사 정보(강사이름·연락처·소속단체·강사번호), 서약 동의 여부
  • 이 정보 중 이름·생년월일·연락처·병력이력·서명은 민감하거나 개인을 식별할 수 있는 정보입니다. 해당 정보는 외부로 전송되지 않으며 사용자 기기에만 머뭅니다.
  • 자동 채우기는 비어 있는 칸만 채우며, 이미 입력된 값은 덮어쓰지 않습니다.
  • 여러 사람이 공용으로 쓰는 브라우저(가족·커플·공용 PC)에서는 저장된 정보가 다음 사용자의 서약서/동의서 화면에 자동으로 채워질 수 있으니 주의하세요. 공용 기기에서는 아래 "저장 정보 삭제 방법"을 참고해 정보를 지우는 것을 권장합니다.

데이터 전송 및 외부 통신

  • 딥스테이션(deepstation.kr): 예약 현황·내역을 조회하기 위해, 사용자가 로그인한 상태에서 딥스테이션의 조회 API(deepstation.kr/rez/ajax.dayinfo.php) 및 예약 상세 페이지(view.php)로 인증된 요청(브라우저 쿠키 포함)을 보냅니다. 이 통신은 사용자의 브라우저와 deepstation.kr 사이에서만 이루어집니다.
  • 파라다이브(paradive.co.kr): 강사 마이페이지 일정을 캘린더로 표시하기 위해 사용자가 로그인한 상태에서 파라다이브 페이지를 읽고, 동의서 페이지에서는 폼 입력을 보조합니다. 이 통신은 사용자의 브라우저와 paradive.co.kr 사이에서만 이루어지며, 동의서 자동 채우기 정보는 외부로 전송되지 않습니다(기기에만 저장).
  • 구글 캘린더(선택 기능): 사용자가 예약/일정을 클릭하면, 해당 예약 정보(날짜·시간·인원·부이·예약번호·전자서명문서 링크 등)를 담은 구글 캘린더 "일정 추가" 링크를 새 탭으로 엽니다. 사용자가 구글 계정으로 일정을 저장하면 이 정보는 구글(제3자)에 전달됩니다. 이 기능은 사용자가 직접 클릭할 때만 동작합니다.
  • Google Analytics(익명 사용량 통계): 어떤 페이지·기능이 얼마나 쓰이는지 파악하기 위해 비식별 사용 정보를 Google Analytics 4(Measurement Protocol)로 전송합니다. 전송 항목은 다음으로 한정됩니다.
    • 방문한 페이지 유형(예: 예약 목록·예약 폼·동의서·파라다이브 캘린더)
    • 사용한 기능(예: 폼 자동 채우기, 구글 캘린더 내보내기, 동의서 자동 채우기)
    • 예약/취소 행동 패턴의 비식별 값(요일·시간대·부수·예약~이용 간격일수·종목·부이 사용 여부)
    • 오류 발생 신호(사이트 구조 변경 등으로 인한 기능 실패 진단용)
    식별자로는 기기에 1회 생성되는 무작위 익명 ID만 사용하며, 이는 어떤 개인정보에서도 파생되지 않습니다. 이름·생년월일·연락처·예약번호·서명 등 개인을 식별할 수 있는 정보는 전송하지 않습니다. 본 통계는 확장 프로그램 설치 사용자에 한정되며 딥스테이션·파라다이브 전체 이용 현황과 무관합니다.
  • 그 외 본 확장 프로그램은 광고 도구를 사용하지 않으며, 위에 명시한 곳(딥스테이션·파라다이브·구글 캘린더·Google Analytics) 외 제3자 서버로 데이터를 전송하지 않습니다.

저장 정보 삭제 방법

현재 확장 프로그램 내 전용 삭제 버튼은 제공되지 않습니다. 저장된 서약서 정보는 다음 방법으로 삭제할 수 있습니다.

  • 확장 프로그램을 제거하면 chrome.storage.local에 저장된 정보도 함께 삭제됩니다.
  • 브라우저의 "인터넷 사용 기록 삭제 → 쿠키 및 기타 사이트 데이터"로 사이트 데이터를 삭제합니다.
  • 세션 캐시(sessionStorage)는 탭 또는 브라우저를 닫으면 자동으로 사라집니다.
